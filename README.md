# Profile Card

## Installation

To use the Profile Card component, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/your-username/profile-card.git
```

2. Navigate to the project directory:

```bash
cd profile-card
```

3. Install the dependencies:

```bash
npm install
```

## Usage

To use the Profile Card component, import it into your React application:

```jsx
import ProfileCard from './components/ProfileCard';

function App() {
  return (
    <div>
      <ProfileCard />
    </div>
  );
}
```

The `ProfileCard` component will render a grid of profile cards with information about each profile, including the name, role, bio, and social media links.

## Testing

To run the tests for the Profile Card component, use the following command:

```bash
npm run dev
```

This will run the test suite and display the results in the console.
